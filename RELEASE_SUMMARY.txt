HOW TO PUBLISH THE RELEASE (After Merging This PR)
================================================================================

QUICKEST METHOD:
  1. Merge this PR to main
  2. git checkout main && git pull
  3. ./scripts/publish-release.sh

ALTERNATIVE - GitHub Web UI:
  1. Visit: https://github.com/BenTade/ethcal-ui/releases/new
  2. Tag: v1.0.0 (create new)
  3. Title: Release v1.0.0
  4. Description: Copy from RELEASE_NOTES.md
  5. Click "Publish release"

ALTERNATIVE - GitHub CLI:
  gh release create v1.0.0 \
    --title "Release v1.0.0" \
    --notes-file RELEASE_NOTES.md \
    --target main
